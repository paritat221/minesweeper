# minesweeper
Minesweeper game for casio fx cg-50 calculator

# how to play
Plug your fx cg-50 into your pc and drag and drop Minesweeper.g3a

# controls 
SHIFT : show cell
ALPHA : flag cell
OPTN : restart
MENU : exit game
